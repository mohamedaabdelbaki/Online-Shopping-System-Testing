# Online Shopping System 

## Project Type
Manual Testing + Bug Reporting

## Tested Website
https://frontend-rey0.onrender.com/

## Objective
To perform end-to-end manual testing on the Online Shopping System,
including requirement analysis, test case design, execution, and bug reporting.
---
## Scope of Testing

### 1. Requirement Analysis
- Explored the website features and user flows.
- Identified core functionalities.

### 2. Test Design
- Created Test Scenarios covering main features.
- Derived detailed Test Cases from scenarios.
- Included both positive and negative test cases.
- Clearly documented assumptions.

### 3. Test Execution
- Executed designed test cases.
- Recorded actual results and test status.

### 4. Bug Reporting
- Documented observed defects in structured Bug Reports.
- Included:
  - Bug ID
  - Title
  - Steps to Reproduce
  -Attachment for the bug
  - Expected Result
  - Actual Result
  - Severity
  - Priority
  - Status
